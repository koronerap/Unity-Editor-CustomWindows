# Unity-Editor-CustomWindows

A collection of handy editor extensions to speed up your Unity workflow.

## 1. Scene Tabs
A browser-style tab bar for navigating between scenes efficiently.

### Features
- **Browser-like Experience:** Switch scenes instantly using tabs.
- **Persistent History:** Remembers your open scenes even after restarting Unity.
- **Quick Add:** Drag & drop scenes or use the **+** button to find any scene in the project.
- **Controls:**
  - `Left Click`: Open scene.
  - `Middle Click` or `x`: Close tab.

**Open via:** `Tape Corps > Scene Tabs`

---

## 2. Quick Select
A persistent "favorites" list for quickly accessing frequently used objects.

### Features
- **Object Bookmarks:** Drag & drop any object (Prefabs, ScriptableObjects, Scene Objects) to the list.
- **One-Click Selection:** Click an item to select and ping it in the Hierarchy or Project view.
- **Editing:** Assign new objects to slots or clear the list easily.

**Open via:** `Tape Corps > Quick Select`
